Intermittent Reminder

  1. Select a time you want the intermitten reminder will start. It must not earlier than the current time.
  2. Select an end time. It must not be earlier than the selected Start Time.
  3. The duration, which is the total hour-difference, is displayed.
  4. Select how many times you want to be notified. Through this, the in-between hours will be calculated and displayed to the users.
  5. Cleck 'Begin' and expect notifications in a moment.
